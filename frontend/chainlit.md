# AI Agent Multi-Tasking System

**Version**: 1.0.0

AI-powered code debugging, analysis, and documentation generation system with multi-task orchestration and MCP integration.

## Features

- **Multi-Language Debugging**: Debug code in Python, JavaScript, TypeScript, Java, Go, Rust, and more
- **Code Analysis**: Analyze code for quality, security, and performance issues
- **Documentation Generation**: Auto-generate documentation in multiple styles
- **Task Orchestration**: Execute multiple AI tasks in sequence or parallel
- **MCP Integration**: Connect to Supabase, Neon, Linear, Notion, and more
- **Conversation History**: Track all agent interactions and execution metrics
- **Local AI Support**: Use Ollama for private, on-device AI processing
- **Cloud Fallback**: Fallback to Mistral AI API for enhanced capabilities

## Usage

1. **Debug Code**: Paste code with an error message to get debugging assistance
2. **Analyze Code**: Submit code for security, performance, or quality analysis
3. **Generate Docs**: Create documentation in Google, NumPy, or Sphinx style
4. **Multi-Task**: Execute multiple analysis tasks in one session
